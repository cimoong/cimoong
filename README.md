## 👨‍💻 About me

11 years writing C#. Long enough to watch `.NET Framework` become `.NET Core` become just `.NET` — same language, three rebrands, zero therapy sessions about it.

By day: backend/full-stack engineer keeping production systems boring (in a good way — boring means nothing's on fire). By night-ish: teaching LLMs to be useful instead of just confident.

- 🔧 Home turf: **C# / ASP.NET Core**, **SQL Server & PostgreSQL**, **Entity Framework**, **Blazor** — the reliable stuff nobody tweets about but everything depends on.
- 🤖 Current obsession: RAG pipelines and LLM agents that answer from *actual* sources instead of vibes, with a human in the loop before anything irreversible happens.
- ☁️ Comfortable end to end — API to database to Azure, plus enough OpenTelemetry to know exactly which line broke at 3am.
- 🧪 Big on citations over hallucinations — if the AI can't point to where it got that from, I don't trust it either.
- 🌍 Remote-friendly, timezone-flexible, allergic to "it works on my machine."

> 💬 Fun fact: my `git commit` messages are more reliable narrators than most news outlets.
